🚀 Quick Start

git clone https://github.com/medissaoui711/apex-core.git
cd apex-core
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python copy_resources.py
python apex.py

💡 First run will prompt you to select a provider (Local/Cloud).
